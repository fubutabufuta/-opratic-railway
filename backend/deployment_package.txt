## 🚀 Railway.app Quick Deploy

### 1. Create GitHub Repository
```bash
# Backend klasöründen çalıştır
git init
git add api/ config/ admin_modern.php .htaccess database_schema_update.sql news_table.sql deploy_guide.md
git commit -m "Production ready backend"
git remote add origin https://github.com/USERNAME/otoasist-backend.git
git push -u origin main
```

### 2. Railway Deployment
1. railway.app → New Project
2. Deploy from GitHub repo
3. Environment variables:
   - DB_HOST: [Railway otomatik]
   - DB_NAME: railway
   - DB_USERNAME: root
   - DB_PASSWORD: [Railway otomatik]

### 3. Database Setup
Railway dashboard → Database → Query → Run:
```sql
-- Schema import
-- database_schema_update.sql içeriğini buraya kopyala

-- News table import  
-- news_table.sql içeriğini buraya kopyala
```
